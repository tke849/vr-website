ezAR FaceDetector Cordova Plugin

Detect faces in a video preview from the VideoOverlay plugin. Uses native OS-level face detection.

Requires the VideoOverlay plugin.

Supported Platforms

  • iOS 7 and greater
  • Android 4.2 and greater
